V0 REGISTER FILE
----------------

General Purpose Registers
-------------------------

R0	zero register - reads return zero, writes are ignored

Caller-Save/Function Argument Registers
---------------------------------------

R0	first function argument, function return value
R1	second function argument, second function return value
R2	third function argument
R3	fourth function argument

Callee-Save/Scratch Registers
-----------------------------

R4	first scratch/local register
R5	second scratch/local register
R6	third scratch/local register
R7	fourth  scratch/local register
R8	fifth scratch/local register
R9	sixth scratch/local register
R10	seventh scratch/local register
R11	eigth scratch/local register
R12/FP	frame-pointer
R12/SP	stack-pointer
R14/LR	link-register (return address)
R15/PC	program counter

Special Registers
-----------------

ID	Description		Contents
--	-----------		--------

MSW     machine status word
TCB     thread control block base address
TLS	thread-local storage pointer
TWC     thread wait channel
IM      interrupt-mask (1-bit enabled)
IV      interrupt vector address
PD      page directory address + flags
IOM     I/O descriptor map base address
BLO     bound-range low limit
BHI     bound-range high limit (base + size)
TLO     timestamp low 32 bits
THI     timestamp high 32 bits
MFR     machine feature word, read-only

MSW Register Bits
-----------------

ID	Description		Bits
--	-----------		----

ZF	zero/equal		8
CF	carry/borrow		1
OF	overflow		2
LT	less than		3
SF	signed			4
IF	interrupt		5

SF	signed result		9
LT	less than		10
GT	greater than		11
IF	interrupts enabled	14
RF	ring-flag, 1 for system 15

MSW/MFR Register Bits
---------------------

ID	Description		Bit
--	-----------		---

MP	multiprocessor		31
PG	paging			30
IO	I/O permission control	29
FP	floating-point unit	28
FM	fixed-point unit	27
GP	graphics unit		26
DP	digital signal unit	25
MD	SIMD unit      		24
VP	vector unit		23

Notes
-----

The ID-field is the hardware thread ID. The control block, i.e. execution
context of the thread can be found at the address

	TCB + ID * TCBSIZE

where TCBSIZE is the size of the processor thread control block
